# Language Switch Helper

This project is for when you forget to change your keyboard language (EN-TH for now).

## Ways to use this software

There are 2 ways to use this software:

1. **AutoHotkey (.ahk script)**

- Use the `th_en.ahk` script with AutoHotkey.
- This script can run when Windows starts up.

2. **Python implementation**

- Use the `th_en.py` script.
- Right now, you need to execute it manually.
